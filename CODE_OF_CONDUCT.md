# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) v2.1.

In short: be respectful, assume good faith, and keep discussion focused on the work. Harassment, discrimination, and personal attacks are not tolerated.

Report unacceptable behavior to the maintainer via [GitHub](https://github.com/ofsazib). Reports are handled confidentially.
